Welcome to Pro Bank

This is a banking application that enable the users to create a new customer, new account, deposit and withdraw funds from an account. To use this application Postman is used.

A new customer,

  • Should be able to register on the system.
  • Should be able to create a new Customer
  • Should be able to create an account
  • Should be able to deposit funds into an account
  • Should be able to withdraw funds from an account
  • Should be able to view the balance of my account
  • Should be able to log out from the system

All the above tasks are achievable by sending HTTP Requests to the end points provided in the postman executable
